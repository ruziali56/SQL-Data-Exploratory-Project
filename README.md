# 📊 SQL Data Exploration Project

## Project Overview

This project focuses on exploring and analyzing a sales database using Microsoft SQL Server. The main objective was to understand the structure of the data, examine key business dimensions and measures, and answer analytical business questions using SQL.

## Objectives

* Explore the database structure
* Analyze dimension tables
* Explore date-related information
* Calculate key business measures
* Analyze data magnitude and distributions
* Identify top and bottom performers
* Practice SQL techniques for business data analysis

## Tools Used

* Microsoft SQL Server
* SQL Server Management Studio (SSMS)

## SQL Skills Demonstrated

* SELECT
* WHERE
* GROUP BY
* ORDER BY
* Aggregate Functions (SUM, COUNT, AVG, MIN, MAX)
* Subqueries
* Window Functions
* TOP
* Common Table Expressions (CTEs)

## Analysis Performed

### Database Exploration

* Explored database tables and their relationships.
* Reviewed the available data for analysis.

### Dimensions Exploration

* Analyzed customer, product, and other dimension data.
* Examined unique values and business categories.

### Date Exploration

* Investigated sales trends over time.
* Analyzed years, months, and other date-based information.

### Measures Exploration

* Calculated key performance metrics such as revenue, quantity sold, and order counts.

### Magnitude Analysis

* Compared sales performance across different products and categories.
* Measured business performance using aggregated metrics.

### Ranking Analysis

* Identified the Top N and Bottom N performing products using SQL ranking functions.

## Key SQL Concepts

* Data Exploration
* Aggregation
* Ranking
* Business Analysis
* Analytical Queries
* Subqueries
* Window Functions

## Conclusion

This project demonstrates practical SQL skills for data exploration and business analysis. It showcases the ability to transform raw database records into meaningful insights using SQL queries, aggregation techniques, subqueries, and window functions.
